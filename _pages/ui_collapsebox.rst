===============
``CollapseBox``
===============

**Inherits from** |ui_layoutbox|_

The ``CollapseBox`` is a ``LayoutBox`` that can be collapsed down to a
label so that advanced options can be hidden until needed.

Properties
----------

In addition to any inherited properties, a ``CollapseBox`` supports:

+---------------+---------------------------------------+-------------+
| Property      | Description                           | Form        |
+===============+=======================================+=============+
| ``collapsed`` | Sets the initial collapsed state of   | boolean     |
|               | the control.                          |             |
+---------------+---------------------------------------+-------------+
| ``label``     | Sets the text that will be displayed  | string      |
|               | in the label bar of the control.      |             |
+---------------+---------------------------------------+-------------+
