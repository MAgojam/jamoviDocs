user manual
===========

   Welcome to the jamovi user-manual. This contains everything you need to know
   about getting up and running for jamovi. Note that there are also `video
   tutorials <https://datalab.cc/jamovi>`__ available from |datalab|, and
   «learning statistics with jamovi» (both as `web documentation 
   <https://lsj.readthedocs.io/en/latest/lsj/Prologue.html>`__ and as 
   `PDF textbook <https://www.learnstatswithjamovi.com/>`__) for those who
   prefer those formats.
   
   If you got stuck or get questions regarding jamovi, these can be asked
   in the `jamovi forum <https://forum.jamovi.org>`__.
